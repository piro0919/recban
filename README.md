# Recban

> Find your ideal band members — Japanese band matching service.

[🔗 Live Site](https://recban.kk-web.link)

## ✨ Features

- 🎸 Musician profile creation
- 🔍 Search & filter by instrument / genre / location
- ✉️ Email notifications via SendGrid
- ⚡ Redis-backed response caching

## 🛠 Tech Stack

- Next.js + React + TypeScript
- SendGrid (email)
- Redis (cache)
- react-hook-form

## 🚀 Development

```bash
pnpm install
pnpm dev
```
